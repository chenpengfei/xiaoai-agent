# 2.2 步骤：音箱端 open-xiaoai client

## 目标

在 LX06 上安装并维护音箱端 client，让它连接 Mac Mini 上的 server：

```text
/data/open-xiaoai/client -> ws://192.168.1.9:4399
```

当前这个步骤已经完成并验证。

## 设置 server 地址

SSH 进入音箱：

```sh
ssh -o HostKeyAlgorithms=+ssh-rsa root@192.168.1.2
```

设置地址：

```sh
mkdir -p /data/open-xiaoai
echo 'ws://192.168.1.9:4399' > /data/open-xiaoai/server.txt
cat /data/open-xiaoai/server.txt
```

## 安装并启动 client

```sh
curl -sSfL https://gitee.com/idootop/artifacts/releases/download/open-xiaoai-client/init.sh -o /tmp/open-xiaoai-init.sh
sh /tmp/open-xiaoai-init.sh
```

安装后应存在：

```text
/data/open-xiaoai/client
/data/open-xiaoai/server.txt
```

## 手动启动 client

```sh
cd /data/open-xiaoai
./client "$(cat server.txt)" >/data/open-xiaoai/client.log 2>&1 &
```

如果需要先清理旧进程：

```sh
killall client 2>/dev/null || true
cd /data/open-xiaoai
(./client "$(cat server.txt)" >/data/open-xiaoai/client.log 2>&1 </dev/null &)
```

## 查看状态

```sh
ps | grep open-xiaoai
netstat -tn | grep 192.168.1.9:4399
tail -f /data/open-xiaoai/client.log
```

成功日志：

```text
✅ 已启动
✅ 已连接: "ws://192.168.1.9:4399"
```

## 配置开机自启动

patched 固件的 `/etc/rc.local` 会调用：

```text
/data/init.sh
```

下载 boot 脚本：

```sh
curl -sSfL -o /data/init.sh https://gitee.com/idootop/artifacts/releases/download/open-xiaoai-client/boot.sh
chmod +x /data/init.sh
```

`/data/init.sh` 会读取：

```text
/data/open-xiaoai/server.txt
```

因此只要 `server.txt` 是：

```text
ws://192.168.1.9:4399
```

音箱重启后就会尝试自动连接 Mac Mini server。

## 注意事项

- 如果 Mac Mini IP 改变，需要同步更新 `/data/open-xiaoai/server.txt`。
- client/server 通信默认缺少强认证和加密设计，只建议局域网使用。
