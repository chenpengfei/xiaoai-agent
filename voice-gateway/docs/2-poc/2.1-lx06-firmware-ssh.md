## SSH 连接

```sh
ssh -o HostKeyAlgorithms=+ssh-rsa root@192.168.1.2
```

如果本机 OpenSSH 还要求开启旧算法：

```sh
ssh   -o HostKeyAlgorithms=+ssh-rsa   -o PubkeyAcceptedAlgorithms=+ssh-rsa   root@192.168.1.2
```

## SSH + mtd 方式

在已经能 SSH 的情况下，也可以通过网络写入 MTD 分区。

```sh
cd /Users/chenpengfei/projects/vibe-coding/xiaoai-agent

dd if=LX06_1.94.13.squashfs bs=1m | ssh -o HostKeyAlgorithms=+ssh-rsa root@192.168.1.2   'mtd unlock system1; mtd -f write - system1; sync'

dd if=LX06_1.94.13_patched.squashfs bs=1m | ssh -o HostKeyAlgorithms=+ssh-rsa root@192.168.1.2   'mtd unlock system0; mtd -f write - system0; sync'
```

验证分区哈希：

```sh
ssh -o HostKeyAlgorithms=+ssh-rsa root@192.168.1.2   "dd if=/dev/mtdblock5 bs=37500699 count=1 2>/dev/null | sha256sum"

ssh -o HostKeyAlgorithms=+ssh-rsa root@192.168.1.2   "dd if=/dev/mtdblock4 bs=38453248 count=1 2>/dev/null | sha256sum"
```

## 注意事项

- 对当前这台 LX06，普通 patched 固件已经验证正常。
- 使用 `mtd write` 时必须确认 IP、分区名和固件文件正确。
- 如果路由器 DHCP 重新分配 IP，优先用 MAC `50:88:11:6f:f2:a8` 找回音箱。
