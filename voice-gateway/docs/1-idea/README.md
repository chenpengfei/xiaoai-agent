# 1-idea

本目录记录 `voice-gateway` 最早期的想法、判断和探索思路。它解释为什么要做这条链路，以及希望把它逐步演进成什么。

上级索引：[Voice Gateway 开发路线](../roadmap.md)

## 原始想法

家里已经有小爱音箱，平时家人和孩子都很常用，是一个高频语音入口。不过目前主要使用的是闹钟这类基础功能。虽然小爱同学最近也接入了小米的大模型，但整体模型能力仍然不足，复杂问题、连续沟通和更个性化的家庭场景还撑不起来。

因此，希望对小爱音箱做改造：保留它作为家里的麦克风和扬声器入口，把这个高频入口接入 Mac Mini 上更聪明的 Agent。

这样一方面可以把连续对话、自然打断、声纹识别这些更接近人类对话的能力引入家庭语音入口；另一方面，通过接入 Hermes 这样的 Agent 生态，可以获得记忆能力，以及和更丰富世界交互的能力。

长期目标不是一次性做出完整系统，而是通过持续迭代，把小爱音箱从一个常用但能力有限的语音入口，逐步演进成家人愿意高频使用的智能管家。

PoC 阶段沉淀出来的知识成果放在 [2-poc/knowledge](../2-poc/knowledge/README.md)。
